# Flask Books Api 

## Set up & Run locally

### 1. Clone the project

```bash
git clone https://github.com/anuraagnagar/flask-books-api.git
```

### 2. Go to the project directory

```bash
cd flask-books-api
```

### 3. Install the Flask Library

```bash
pip install flask
```

### 4. Run the server

```bash
python app.py
```
